## Objetivo
Analizar las ventas semanales de Walmart mediante modelos de Machine Learning para apoyar la toma de decisiones empresariales.

## Dataset
Walmart Sales Dataset

## Modelos utilizados
- Random Forest Regressor
- K-Means Clustering

## Herramientas utilizadas
- Python
- Google Colab
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Resultados principales
El modelo predictivo permitió estimar las ventas semanales utilizando variables económicas y operativas.

El modelo K-Means permitió identificar grupos de tiendas con comportamientos similares.
